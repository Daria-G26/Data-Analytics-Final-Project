## Overview
This project analyzes well-being indicators across European countries, comparing cognitive life evaluation (Cantril ladder score) with emotional happiness.
The analysis includes data cleaning, visualizations, correlation studies, and interpretation of findings.
## Included Files
* notebook/Activity vs Well-being.ipynb — main analysis and code

* data/insufficient_physical_activity.csv — raw dataset

* data/happiness_and_life_satisfaction.csv — raw dataset

* slides/Rational vs. Emotional Well-Being A European Paradox.pptx — final presentation
